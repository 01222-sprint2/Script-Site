Cotton M20

Nosso produto consiste em um sensor conectado a uma placa de Arduino, esse que faz a captura dos dados de temperatura e umidade do ambiente externo. Assim enviando para a plataforma digital, a mesma que disponibilizará uma dashboard para os clientes cadastrado. Assim tornando possível a visualização dos dados e de alertas para o cotonicultor conseguir tomar as melhores decisões para o seu negócio.

MAPEAMENTO

Análise do contexto de negócio do cliente e Arquitetura do projeto

IMPLANTAÇÃO

Implementação dos sensores dispostos de forma estrátegica pelo hectare

ALERTAS

Alertas presentes na plataforma digital, para o monitoramento de qualquer lugar.
